# Morzh-RDS-Encoder
Program for generating RDS signal without any additional hardware.

Super easy and understandable python program that let's you generate RDS signal and send it to any sound output.
# How to use
First open the program and choose the output (It will be on the top).
Then write your PS(Station Name) and PTY(Rock, Pop etc.)
It is also important to generate your PI hex(You can do that at https://db.wtfda.org/rds2.html by entering your call sign or name into the call sign field)
After that choose you NowPlaying.txt file or any file that has your Artist - Song title or any other text you want to display on your radio station.
After you played around with program click start.
Note: If you don't want to loose your configuration click save config button and now every time you load you will have the same configurations.
